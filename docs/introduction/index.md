# はじめに

## このコンテンツについて

色彩検定 UC 級（色のユニバーサルデザイン級）の合格を目指して学習するためのコンテンツです。

UC 級は、色覚の多様性に配慮した「色のユニバーサルデザイン」に関する知識を問う級です。年齢・性別・色覚特性などの違いにかかわらず、できるだけ多くの人に正しく情報が伝わる色使いを学びます。

このコンテンツでは、検定範囲をレッスン単位に分けて、少しずつ知識を積み上げていけるように整理していきます。

## 進め方

- 1 レッスンずつ順番に読み進めてください
- 暗記よりも「概念をつかむ」ことを優先しています
- 検定対策として、要点・用語・配色例を意識して読むのがおすすめです
