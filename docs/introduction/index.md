# はじめに

## このコンテンツについて

色彩検定 UC 級（色のユニバーサルデザイン級）の合格を目指して学習するためのコンテンツです。

UC 級は、色覚の多様性に配慮した「色のユニバーサルデザイン」に関する知識を問う級です。年齢・性別・色覚特性などの違いにかかわらず、できるだけ多くの人に正しく情報が伝わる色使いを学びます。

全 6 章 28 レッスンで構成し、1 レッスン 1 トピックで段階的に学べるようにしています。

## UC 級の概要

| 項目 | 内容 |
|------|------|
| 実施回数 | 年 2 回（夏期・冬期） |
| 試験時間 | 60 分 |
| 出題形式 | 4 択マークシート（一部記述） |
| 満点 | 200 点 |
| 合格ライン | 概ね 7 割（年度により変動） |
| 受験資格 | 制限なし（誰でも受験可能） |

近年の合格率は 8 割前後と、色彩検定の他の級と比べても比較的取り組みやすい級です。ただし、出題範囲を確実に押さえることが合格への近道です。

## このコンテンツの特徴

- **全 28 レッスン**で全 6 章をカバー
- **1 レッスン 15 分程度**で読める分量
- **図解・表・Mermaid 図**を多用して視覚的に理解できる
- **「試験のポイント」**で各レッスンの暗記すべき要点を整理
- **ドリル**で章別・ランダム・復習に対応。回答履歴はブラウザに保存
- **当事者の視点**を尊重し、「色覚異常」ではなく「色覚特性」「色覚の多様性」を基本表記に統一

## 学習の進め方（おすすめ）

### 1. レッスンを順番に読み進める（所要時間: 約 7〜10 時間）

レッスンは前のレッスンの内容を踏まえて進む構成です。1 日に 3〜5 レッスンずつ読み進めると、1 週間ほどで一周できます。

### 2. 各章のドリルで理解度を確認

レッスンを読み終えたら、対応する章のドリルを解いて理解度を確かめましょう。間違えた問題は[復習機能](/quiz/review/)でまとめて解き直せます。

### 3. 試験前にランダム出題で総仕上げ

試験 1 週間前を目安に、[ランダム出題（全問）](/quiz/random/)で全範囲をチェックします。

::: tip 暗記すべき重要数値
試験で頻出の数値です。これらは確実に覚えておきましょう。

- **可視光線**: 約 380〜780nm
- **色覚特性者の割合**: 男性 約 5%（20 人に 1 人）、女性 約 0.2%（500 人に 1 人）
- **コントラスト比**: 標準テキスト 4.5:1 以上、大きな文字・図形 3:1 以上（WCAG AA）。白黒の最大は 21:1
- **錐体の対応**: P型（1型） = L錐体、D型（2型） = M錐体、T型（3型） = S錐体
:::

## 用語について

本コンテンツでは、当事者の方々への配慮として以下の方針で用語を統一しています。

- **「色覚特性」「色覚の多様性」**を基本表記とします。「色覚異常」は医学用語として説明する文脈でのみ使用します
- **「色盲」「色弱」**は歴史的経緯の説明を除き、原則として使用しません
- **「P型（1型）」「D型（2型）」「T型（3型）」「A型」**の表記を基本とします

色のユニバーサルデザインは、「特定の人を救済する」取り組みではなく、「すべての人にとってより良い情報伝達を実現する」取り組みです。この姿勢を学習を通じて自然に身につけていただければ幸いです。
