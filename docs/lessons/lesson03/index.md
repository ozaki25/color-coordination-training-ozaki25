# lesson03: 視覚情報にかかわるユニバーサルデザイン

## このレッスンで学ぶこと

- 視覚情報のユニバーサルデザインが対象とする範囲を理解する
- 色だけでなく、形・文字・配置・コントラストなど視覚情報全体としてのUDの考え方を把握する
- 色のUDが視覚情報UDの一部であり、他の手段と組み合わせて初めて機能することを理解する
- 「誰も排除しない設計」の出発点としての視覚情報UDの位置づけを学ぶ

## 視覚情報のUDとは

視覚情報のユニバーサルデザイン（UD）は、見ることに関わるあらゆる情報伝達を、できるだけ多くの人にとって伝わるように設計する考え方です。色覚特性のある人だけでなく、高齢者、弱視者、外国人、子ども、明るすぎる屋外や薄暗い室内で見る人など、さまざまな状況の人が対象になります。

色は情報を伝える強力な手段ですが、色だけに頼ると、色覚特性のある人や白黒印刷で読む人には情報が届きません。視覚情報のUDでは、色を含めた**複数の手がかり**を組み合わせて情報を伝えます。

## 視覚情報の主な手がかり

| 手がかり | 例 |
|---------|---|
| 色 | 区別・強調・分類 |
| 形・記号 | 信号機の灯の並び、警告マーク、ピクトグラム |
| 文字・テキスト | ラベル、見出し、注釈 |
| 配置・位置 | 縦型信号機の上下、左右の対比 |
| 明度・コントラスト | 背景と文字の差、明るさの階調 |
| パターン・テクスチャ | ハッチング、点描、地紋 |

色のUDは、これらの手がかりの中で「色」をできるだけ多くの人に伝わるよう整える取り組みです。一方で、情報を確実に届けるには、**色以外の手がかりも併用する**のが基本です。

## 色のUDと視覚情報UDの関係

色のUDは視覚情報UDの一部です。色のUDがうまく機能するためには、色以外の手がかりも適切に設計されている必要があります。逆に、色だけを工夫しても他の手がかり（文字の大きさやコントラスト）が不十分なら、伝わるべき情報が伝わりません。

::: tip 「色のUD」は単独では完結しない
色をきれいに整えても、文字が小さすぎたり、背景とのコントラストが弱かったりすれば情報は伝わりません。色のUDは「視覚情報UD」という大きな枠組みの中の一部として理解しましょう。
:::

## キーワード

| 用語 | 説明 |
|------|------|
| 視覚情報のUD | 色だけでなく形・文字・配置・コントラストなど、見ることに関わる情報全般のUD |
| 複数の手がかり | 色・形・文字・配置・パターンなどを組み合わせて情報を伝える考え方 |
| 色のUDと視覚情報UDの関係 | 色のUDは視覚情報UDの一部。他の手がかりと組み合わせて機能する |

## 試験のポイント

- 視覚情報のUDは**色だけでなく、形・文字・配置・コントラスト・パターン**などの手がかりを含む
- 色は強力な情報伝達手段だが、**色だけに頼らない**ことがUDの基本
- **色のUDは視覚情報UDの一部**であり、他の手がかりと組み合わせて機能する
