# lesson05: 光源色と物体色 — 分光反射率曲線

## このレッスンで学ぶこと

- 光源色と物体色の違いを説明できる
- 物体の色は「光・物体・視覚」の3つで決まることを理解する
- 分光反射率（透過率）曲線の読み方を覚える
- 同じ物体でも光源が変わると色の見え方が変わる理由を理解する

## 色の見え方の3要素

物体の色は、次の3つが揃って初めて見えます。

1. **光源**：光を発するもの（太陽・蛍光灯・LED・ろうそく など）
2. **物体**：光を反射・吸収・透過するもの
3. **視覚（眼と脳）**：受け取った光を「色」として解釈する

このうちどれか1つでも欠けると色は見えません。たとえば暗闇では光源がないため色が見えず、影に隠れた物体も同じです。

## 光源色と物体色

| 種類 | 説明 | 例 |
|------|------|---|
| **光源色** | 光源そのものから発される色 | 太陽の白光、ネオン管の青、LEDの色 |
| **物体色** | 物体が光を反射（または透過）した結果見える色 | リンゴの赤、葉の緑、青いガラス越しの景色 |

ふだん私たちが「色」と呼ぶもののほとんどは**物体色**です。光源色は光そのものを直接見たときに知覚されます。

## 物体が色を持つしくみ

物体は、当たった光のうち**ある波長を反射し、別の波長を吸収する**性質をもちます。反射された光が眼に届き、その波長構成によって色が知覚されます。

- 赤いリンゴ：長波長（赤系）の光を強く反射し、それ以外を吸収する
- 緑の葉：中波長（緑系）の光を強く反射する
- 黒い物体：ほとんどの波長を吸収する → 反射光がほぼないので「黒」に見える
- 白い物体：すべての波長をほぼ等しく反射する

## 分光反射（透過）率曲線

物体が**どの波長の光をどれだけ反射するか**を、波長を横軸・反射率を縦軸にとってグラフにしたものを**分光反射率曲線**といいます。透過する物体（ガラスやフィルム）の場合は**分光透過率曲線**となります。

- グラフが**長波長側で高い** → 赤系の物体
- グラフが**中波長側で高い** → 緑系の物体
- グラフが**短波長側で高い** → 青系の物体
- **全体が平坦** → 無彩色（白・グレー・黒は反射率の高低で決まる）

::: tip 物体色を決めるのは「光×物体の分光特性×視覚」
分光反射率曲線は物体の固有の性質ですが、実際に見える色は「光源の分光分布 × 物体の分光反射率 × 視覚の感度」の組み合わせで決まります。
:::

## 光源が変わると色も変わる（演色性）

同じ赤いリンゴでも、太陽光の下と蛍光灯の下では見え方が変わります。これは**光源によって含まれる波長の構成（分光分布）が違う**ためです。「光源 × 物体 = 反射光」なので、光源が変われば反射光も変わります。

光源が物体の色をどれだけ自然に見せるかを示す指標を**演色性**といい、これは色のUDを実践するうえでも重要な概念です。

::: info 色の恒常性
光源が変わると反射光の波長構成は物理的に変化していますが、私たちは「白い紙は白い紙」「赤いリンゴは赤いリンゴ」とほぼ同じ色に知覚します。脳が照明条件を補正してくれるこの働きを**色の恒常性（color constancy）**といいます。完全ではないため、強い色被りのある照明下では見え方が崩れます。
:::

## キーワード

| 用語 | 説明 |
|------|------|
| 光源色 | 光源そのものから発される色 |
| 物体色 | 物体が光を反射・透過した結果見える色 |
| 色の3要素 | 光源・物体・視覚（眼と脳） |
| 分光反射率曲線 | 物体が波長ごとにどれだけ反射するかを示すグラフ |
| 分光透過率曲線 | 透過する物体について、波長ごとの透過率を示すグラフ |
| 演色性 | 光源が物体の色をどれだけ自然に見せるかの指標 |
| 色の恒常性 | 光源が変わっても物体色をほぼ同じと知覚する脳の働き |

## 試験のポイント

- 色は **光源・物体・視覚** の3つが揃って初めて見える
- **光源色**＝光そのものの色、**物体色**＝物体が反射・透過した結果見える色
- 物体は **特定の波長を反射し、別の波長を吸収する**ことで色を持つ
- **分光反射率曲線**：横軸＝波長、縦軸＝反射率。曲線の形で物体の色が分かる
- 光源が変わると物体の見え方も変わる（演色性）
