# lesson12: 色名 — 言葉で色を伝える

## このレッスンで学ぶこと

- **系統色名**と**慣用色名**の違いと使い分けを理解する
- JIS（日本産業規格）で定められている基本色名・系統色名の仕組みを知る
- 色名で色を伝えることの利点と限界を整理する
- 色のUDの観点で「色名を付記する」ことが重要な理由を理解する

## 色を「言葉」で表す方法

色は数値（マンセル値・PCCS のトーンなど）で正確に表せますが、日常で人に色を伝えるときは**言葉（色名）**を使うのが普通です。

色名は大きく分けて 2 種類あります。

- **系統色名**: 修飾語と基本色名を組み合わせて色を表す。誰でも体系的に理解できる
- **慣用色名**: 自然物や染料に由来する固有の名前。直感的で文化的な響きを持つ

どちらも色を伝える役割は同じですが、性格が大きく異なります。

## 系統色名

**系統色名**は、ものさしのように決められた基本色名に、明度・彩度・色相を表す修飾語を付けて色を表す方式です。JIS Z 8102「物体色の色名」で体系が定められています。

### 基本色名（JIS）

JIS では、有彩色 10 種と無彩色 3 種、合計 13 個の**基本色名**が定められています。

| 区分 | 基本色名 |
|------|---------|
| 有彩色 | 赤 / 黄赤 / 黄 / 黄緑 / 緑 / 青緑 / 青 / 青紫 / 紫 / 赤紫 |
| 無彩色 | 白 / 灰色 / 黒 |

### 修飾語

基本色名の前に**修飾語**を付けて、明度・彩度・色相のニュアンスを加えます。

- 明度・彩度の修飾語: 「ごく薄い」「薄い」「明るい」「鮮やかな」「濃い」「暗い」など
- 色相の修飾語: 「赤みの」「黄みの」「緑みの」「青みの」など

たとえば「**鮮やかな赤**」「**暗い青緑**」「**黄みの白**」のように表します。修飾語と基本色名を組み合わせるため、色の位置（色相・明度・彩度）が言葉から推測できます。

::: tip 系統色名の長所
**規則的**なので、初めて聞く色名でも「だいたいどんな色か」が推測できます。たとえば「暗い赤紫」と聞けば、赤紫を暗くした色だと分かります。
:::

## 慣用色名

**慣用色名**は、自然物・染料・地名などに由来する固有の名前です。古くから使われ、文化的・情緒的な響きを持ちます。

| 慣用色名 | 由来 | だいたいの色 |
|---------|------|-------------|
| 桜色 | 桜の花びら | 淡い赤 |
| 若草色 | 春に芽吹く草 | 明るい黄緑 |
| 朱色 | 朱（鉱物） | 鮮やかな黄赤 |
| 群青 | 鉱物顔料 | 濃い青 |
| あずき色 | 小豆 | 暗い赤 |
| 山吹色 | ヤマブキの花 | 明るい黄 |

JIS でも代表的な慣用色名が**JIS 慣用色名**として規格化されており、対応する系統色名や色票値が示されています。

::: warning 慣用色名は人によってイメージが揺れやすい
「桜色」と言っても、人によって思い浮かべる色は少しずつ違います。**厳密に伝えたいときは系統色名や数値（マンセル・PCCS）を併用**するのが安全です。
:::

## 系統色名と慣用色名の比較

| 観点 | 系統色名 | 慣用色名 |
|------|---------|---------|
| 表し方 | 修飾語＋基本色名 | 固有の名前 |
| 規則性 | あり（推測しやすい） | なし（覚えるしかない） |
| 直感性 | やや弱い | 強い |
| 文化的背景 | 中立 | 国・時代に依存 |
| 用途の例 | 設計図書、技術文書 | 商品名、ファッション、季節表現 |

両者は対立するものではなく、**場面に応じて使い分け**ます。技術的な仕様には系統色名、印象を伝える表現には慣用色名、というのが基本です。

## 色名で伝える利点と限界

### 利点

- **直感的**: 数値より分かりやすく、専門知識がなくても伝わる
- **共有しやすい**: 会話・文書・標識の説明文など、あらゆる場面で使える
- **色覚に依存しない**: 色そのものが見えにくい人にも、言葉なら伝えられる

### 限界

- **個人差がある**: 同じ色名でも、人が思い浮かべる色は揺れる
- **印刷・画面で再現がぶれる**: 色名だけでは厳密な色を指定できない
- **慣用色名は文化依存**: 国や世代によって通じないことがある

厳密な指定が必要な場合は、色名と**マンセル値や色票番号**を併用します。色名は「人に伝える」、数値は「機械や工程に伝える」と整理すると分かりやすいです。

## 色のUDにおける色名の重要性

色のUDでは、**「色だけで情報を伝えない」**ことが基本原則です（[lesson03](/lessons/lesson03/) 参照）。その実践方法のひとつが**「色名を付記する」**ことです。

::: info 色名付記の具体例
- 路線図: 駅名の脇に「赤色：A 線」と書き添える
- グラフ: 凡例に色見本だけでなく「青：売上、橙：利益」と書く
- カラーペン: 軸に「赤」「青」と色名を印字する
- 服のタグ: 色番号と色名を併記する
:::

色名が付いていれば、色を見分けにくい人でも**言葉から情報を取り出せます**。これは色覚特性者だけでなく、白内障で色がくすんで見える高齢者（[lesson23](/lessons/lesson23/)）や、暗い場所・小さな面積で色を判別しにくい場面でも役立ちます。

色のUDの設計修正ポイント（[lesson28](/lessons/lesson28/)）でも、「色名を加える」は代表的な改善手法として挙げられます。

## キーワード

| 用語 | 説明 |
|------|------|
| 系統色名 | 修飾語と基本色名を組み合わせて色を表す方式。JIS Z 8102 で規定 |
| 慣用色名 | 自然物・染料などに由来する固有の色名（桜色・若草色など） |
| 基本色名 | JIS で定められた色名の核。有彩色 10 種＋無彩色 3 種の計 13 個 |
| 修飾語 | 基本色名に付けて明度・彩度・色相のニュアンスを表す語（鮮やかな、暗い、赤みの など） |
| JIS Z 8102 | 物体色の色名を定めた日本産業規格 |
| 色名の付記 | 色のUDで、色見本に色名を書き添えて色だけに依存させない手法 |

## 試験のポイント

- 系統色名は「修飾語＋基本色名」、慣用色名は「固有の名前」と区別する
- JIS の**基本色名は有彩色 10＋無彩色 3＝13 個**であることを覚える
- 有彩色の基本色名（赤・黄赤・黄・黄緑・緑・青緑・青・青紫・紫・赤紫）の順序は色相環に沿う
- 系統色名は**規則的で推測しやすい**、慣用色名は**直感的だが個人差がある**
- 慣用色名の代表例（桜色・若草色・朱色・群青・山吹色 など）とおおよその色を結び付けられるようにする
- 色のUDでは**「色名を付記する」**ことが、色だけに依存しない設計の代表的手法
