# lesson17: 色の誤認 — 色の呼び間違いはなぜ起きるか

## このレッスンで学ぶこと

- 色覚タイプごとに色をどう呼ぶ傾向があるかを理解する
- 「赤を茶や緑と呼ぶ」など代表的な色誤認パターンを知る
- 色誤認が起こりやすい**条件**（面積・照度・彩度・隣接色）を把握する
- 経験や文脈が色名呼称を補正することを理解する
- UD の観点で「色名を付記する」「色だけに頼らない」設計が必要な根拠を確認する

## 「色の誤認」とは

**色の誤認**とは、ある色を別の色と取り違えたり、本来とは違う色名で呼んでしまうことを指します。

C型（標準）の人でも、暗い場所や小さな面積では色を間違えることがあります。色覚特性者（P型・D型・T型）では、混同色線上に並ぶ色（[lesson16](/lessons/lesson16/) 参照）が同じ色に見えるため、より起こりやすくなります。

色の誤認は「目の前の色」を間違えるだけでなく、**色を言葉に変換する段階（色名呼称）でずれる**ことも含みます。UD では、この呼称ずれを前提として設計することが重要です。

## 色覚タイプと色名呼称の傾向

P型・D型は L 錐体・M 錐体の働き方が C型と異なるため、赤と緑の差を感じにくく、独特の呼称傾向が見られます。

### P型（1型）の傾向

P型は **L 錐体**の異常で、長波長側（赤）の感度が低くなります。

- **赤**: 暗く感じられ、「**茶**」や「**黒っぽい色**」と呼ばれることがある
- **緑**: 「**黄色っぽい緑**」「**茶**」と呼ばれることがある
- **赤と黒**、**赤と濃い緑**、**ピンクと水色**などが混同しやすい

### D型（2型）の傾向

D型は **M 錐体**の異常で、中波長側（緑）の感度が低くなります。

- **赤**: 明るさはあまり下がらず、「**緑**」「**茶**」「**橙**」などと呼ばれることがある
- **緑**: 「**赤**」「**茶**」「**黄**」と呼ばれることがある
- **赤と緑**、**橙と黄緑**、**ピンクと灰色**などが混同しやすい

::: info P型と D型はどちらも「赤緑が分かりにくい」
P型・D型は同じ系統の混同を起こしますが、P型は**赤が暗く見える**点が特徴的です。一方、D型はその傾向が弱く、明るさからは赤と緑を区別しにくいです。詳しくは [lesson14](/lessons/lesson14/)・[lesson15](/lessons/lesson15/) を参照してください。
:::

### T型（3型）の傾向

T型は **S 錐体**の異常で、短波長側（青）の感度が低くなります。

- **青と緑**、**黄と白（やピンク）** などが混同しやすい
- 「青空が緑がかって見える」などの体験談がある

T型は先天性ではきわめてまれですが、加齢や眼疾患による後天性の変化として現れることがあります（[lesson21](/lessons/lesson21/)・[lesson23](/lessons/lesson23/) 参照）。

## 代表的な色誤認パターン

赤緑系（P型・D型）で混同されやすい代表的な組み合わせを示します。

| 元の色 | 呼ばれやすい色 |
|--------|--------------|
| 赤 | 茶・黒・緑 |
| 緑 | 茶・黄・赤 |
| 橙 | 黄緑・黄 |
| ピンク | 灰色・水色 |
| 黄緑 | 橙・黄 |
| 紫 | 青 |

T型・後天性で混同されやすい組み合わせの代表例。

| 元の色 | 呼ばれやすい色 |
|--------|--------------|
| 青 | 緑 |
| 黄 | 白・ピンク |

これらは「いつも必ず間違える」わけではなく、後述の**条件**によって起こりやすさが変わります。

## 色誤認が起こりやすい条件

C型・色覚特性者を問わず、次のような条件では色誤認が起こりやすくなります。色のUDではこれらを避けるのが基本方針です。

### 1. 小さい面積

色は面積が大きいほど鮮やかにはっきり見え、小さいほど薄く・暗く・グレーがかって見えます（**面積効果**）。

- 細い線・小さな点・小さな文字は色が伝わりにくい
- 路線図の細い線、グラフの細い折れ線、地図の小さな県境などが該当する

### 2. 低照度（暗い場所）

暗い場所では錐体の働きが弱まり、桿体（[lesson07](/lessons/lesson07/) 参照）が優位になります。

- 色の差が分かりにくくなる
- 特に赤が暗く沈んで見える（**プルキンエ現象**）
- 夜間の屋外、間接照明のレストラン、停電時の非常表示など

### 3. 低彩度

彩度が低い色（くすんだ色・パステル調・灰色みのある色）は色相の差が縮まり、色名呼称が揺れます。

- 似た明度の低彩度色どうしは、C型でも区別しにくい
- 色覚特性者ではさらに混同が増える

### 4. 隣接色との関係（色の対比）

隣り合う色の影響で、本来の色とは違って見える現象（**色の対比**）が起こります。

- 赤の中の灰色は緑がかって見える（補色対比）
- 明るい背景の中の中間色は暗く見える（明度対比）
- 背景色との明度差・彩度差が小さいと、図と地が分離せず色が判別しにくい

### 5. 観察距離・視力

遠くから見たり視力が低い状態では、細部の色情報が失われます。高齢者や弱視（[lesson24](/lessons/lesson24/)）でも同様の影響が出ます。

::: warning 条件が重なると誤認は急増する
「小さく」「暗く」「くすんだ色」を「似た色の上に置く」と、C型でも誤認します。色覚特性者では決定的に見分けがつかなくなります。掲示物・サイン・グラフを設計するときは、これらの条件が**重ならない**ようにします。
:::

## 経験と文脈による補正

色覚特性者は、日常生活の中で**経験と文脈**から色を推測しています。

- 信号機: 位置（左が青・中央が黄・右が赤）で判断する
- 信号の点滅: 横型信号で点滅している側＝注意（黄）と知る
- 食材の状態: 肉の焼け具合は色だけでなく**時間と温度**で判断する
- 服の色: 買ったときの色を**記憶**しておく

このような補正は、慣れた環境では有効に働きますが、**初めての場面**や**情報の手がかりが色だけの場面**では補正できません。だからこそ、色のUDでは色だけに頼らない手がかり（形・記号・色名・位置）が必要になります。

## UD への示唆

色誤認の傾向は、色のUDで取るべき対策を直接的に示しています。

- **色名を付記する**: 「赤：危険」「青：安全」と書き添える（[lesson12](/lessons/lesson12/) 参照）
- **色だけに頼らない**: 形・パターン・記号・位置・テキストを併用する
- **混同しやすい組み合わせを避ける**: 赤と緑、赤と黒、ピンクと灰色 など
- **明度差を確保する**: 色相の差ではなく**明度差**で見分けられるようにする
- **十分な面積・コントラスト**: 細い線・小さな点を色だけで区別させない

これらは色のUDの設計手順（[lesson26](/lessons/lesson26/)）や設計修正 4 ポイント（[lesson28](/lessons/lesson28/)）でも繰り返し登場する基本原則です。

## キーワード

| 用語 | 説明 |
|------|------|
| 色の誤認 | 色を別の色と取り違えたり、違う色名で呼んでしまうこと |
| 色名呼称 | 見えている色を言葉（色名）に変換する行為 |
| 面積効果 | 色は面積が大きいほど鮮やかに、小さいほど淡く見える現象 |
| 低彩度 | 彩度が低くくすんだ色。色相の差が縮まり誤認が起きやすい |
| 色の対比 | 隣接色の影響で本来とは違って見える現象 |
| プルキンエ現象 | 暗所で青が相対的に明るく、赤が暗く感じられる現象 |

## 試験のポイント

- P型は**赤が暗く沈んで**見え、「茶」「黒」と呼ばれることがある
- D型は赤と緑の明るさが似て、互いに混同しやすい
- T型では**青と緑**、**黄と白**が混同しやすい
- 色誤認が起こりやすい条件 = **小さい面積・低照度・低彩度・隣接色との影響**
- 当事者は**経験と文脈で補正**するが、初めての場面では補正できない
- UDの基本対策 = **色名付記・色以外の手がかり併用・混同色の回避・明度差確保**
- 「赤」を必ず「茶」と呼ぶわけではない。条件次第で呼称が変わる点に注意
