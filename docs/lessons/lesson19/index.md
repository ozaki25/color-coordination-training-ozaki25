# lesson19: 色覚検査法 — 自分の色覚を知る

## このレッスンで学ぶこと

- 代表的な色覚検査の種類と仕組みを理解する
- **石原色覚検査表**（仮性同色表）の目的と読み取り方を知る
- **パネル D-15 テスト**で色相配列を測ることの意味を理解する
- **アノマロスコープ**による精密検査の位置づけを知る
- 学校健診での色覚検査の歴史（廃止→再開）を整理する
- UD 観点での意義：「自分の色覚を知る」ことが配慮の出発点であることを理解する

## 検査の目的

色覚検査の目的は、**「異常を診断する」ことではなく、自分の色覚特性を知ること**にあります。色覚特性は治療する病気ではなく、見え方の個性のひとつです。検査で得られる情報は、進学・就職・日常生活で**自分に合った工夫**をするための材料になります。

代表的な検査は次の 3 種類です。

| 検査 | 目的 | 検査内容の概要 |
|------|------|--------------|
| 石原色覚検査表 | スクリーニング | 色の点描の中に数字や図形を読み取る |
| パネル D-15 | タイプ分類・程度判定 | 色相に沿って色票を並べる |
| アノマロスコープ | 精密分類 | 赤緑の混色比を調整して黄色に合わせる |

## 石原色覚検査表（仮性同色表）

**石原色覚検査表**は、日本人医師・石原忍が考案した世界で最も普及した色覚検査表です。**仮性同色表（ぎせいどうしょくひょう）**とも呼ばれます。

### 仕組み

検査表のページには、明度・彩度の異なる**多くの色の小さな点**が並んでいます。その中に、**特定の色**で数字や図形が描かれています。

- **C型**の人は、色の違いで数字や図形を**読み取れる**
- **色覚特性者**は、特定の色が**地の色に紛れて見えにくい**ため、別の数字に見えたり読めなかったりする

このように、**色の違いを利用して数字を浮き上がらせる**仕組みです。

### 検査の進め方

- 表を 75cm 程度の距離で見る
- 自然光に近い照明で（蛍光灯や白熱灯では結果がずれることがある）
- 各ページを **3 秒以内**に読む（じっくり見ると正解してしまう）

### 位置づけ

石原表は**スクリーニング（ふるい分け）**のための検査です。「色覚特性の可能性があるかどうか」は分かりますが、**タイプ（P型・D型・T型）の正確な分類まではできません**。詳しいタイプ判定は次のパネル D-15 やアノマロスコープに進みます。

::: info 「仮性同色表」の意味
**色覚特性者には複数の色が同じに見える**ことを利用して作られた表、という意味です。「仮性」は「みかけの」という意味で、本来は違う色なのに**特定の人には同じ色に見える**ことを指します。
:::

## パネル D-15 テスト

**パネル D-15 テスト**（ファンスワース パネル D-15）は、15 個の色票を**色相の順番に並べる**検査です。「D」は Dichotomous（二分法）、「15」は色票の数を表します。

### 仕組み

- 基準色（パイロット色）の隣に、似た色の色票を順に並べる
- 完成すると色相環の一部のような順番に並ぶはず
- 色覚特性者は混同色線上の色を取り違えるため、特徴的な**並び替えパターン**を示す

### 並び替えパターンから分かること

完成した配列を線でつなぐと、**混同色線**（[lesson16](/lessons/lesson16/) 参照）の方向に沿って交差線が現れます。この交差の方向によって P型・D型・T型を見分けます。

| 並び替えパターン | 示唆 |
|----------------|------|
| 順番通り（軽微な乱れのみ） | 異常なしまたは弱度 |
| 赤緑方向の交差 | P型 または D型 |
| 青黄方向の交差 | T型 |

### 位置づけ

パネル D-15 は**程度判定**（pass / fail）と**タイプの目安**を得るための検査です。職業適性の判定（鉄道・航空関係など）にも使われます。

## アノマロスコープ

**アノマロスコープ**は、色覚を最も精密に検査できる装置です。

### 仕組み

検査者は接眼レンズをのぞき、視野内の上下に分かれた色を見ます。

- **上半分**: 赤と緑の混合光（混色比を変えられる）
- **下半分**: 黄色の単色光（明るさを変えられる）

被検者は、上半分が下半分と**完全に同じ黄色**に見えるように、赤と緑の混合比と黄色の明るさを調整します。

### 何が分かるか

C型では赤と緑をある一定の比でしか混ぜないと黄色に合いません。一方、**色覚特性者ではその比率や許容範囲が異なる**ため、装置の数値から **P型・D型の判別と程度**を厳密に求められます。

| タイプ | 特徴 |
|--------|------|
| C型 | 決まった比率でのみ一致 |
| 強度 P型・D型 | どんな赤緑比でも、明るさ調整さえできれば一致してしまう |
| 弱度 P型・D型 | 一致する比率の幅が C型よりも広い |

### 位置づけ

アノマロスコープは**確定診断**に使われる検査です。装置が高価で扱いに専門知識が必要なため、眼科などの専門機関で実施されます。

## 学校健診での色覚検査の歴史

日本では学校での色覚検査が長く実施されてきましたが、社会的な議論を経て扱いが変わってきました。

| 時期 | 主な動き |
|------|---------|
| 戦後〜1990 年代 | 学校で全員に色覚検査を実施 |
| 1995 年 | 就職時の色覚制限が大幅に緩和される |
| 2003 年 | **学校健診での必須項目から削除**（プライバシー配慮・差別防止のため） |
| 2014 年 | 自分の色覚特性を知らずに進学・就職で困る例が報告され、**希望者に対する検査の実施**が推奨される |

::: warning 「検査を受けない」リスク
2003 年の削除以降、自分の色覚特性を**知らないまま大人になる**人が増えました。進学・就職の段階で初めて知り、希望進路を変えざるを得なくなる例も報告されています。現在は**希望者には学校で検査を受けられる**ように見直されています。
:::

## UD 観点での意義

色覚検査の本当の意義は、**「異常を見つけて排除する」ことではなく、「自分の色覚を知り、必要な工夫や配慮を考えるきっかけにする」**ことにあります。

- 当事者本人にとって: 自分の見え方を理解し、進学・就職・日常生活で工夫できる
- 周囲・社会にとって: 色覚特性者の見え方を知り、配慮した情報伝達を設計できる
- 設計者にとって: 検査の仕組み（混同される色のパターン）を理解することで、混同しにくい配色（[lesson20](/lessons/lesson20/)〜[lesson22](/lessons/lesson22/)）を選べる

色覚検査は当事者だけのものではなく、**色のUDを進める社会全体の出発点**です。

## 試験のポイント

- **石原色覚検査表 = 仮性同色表**で、**スクリーニング**目的。タイプ判定までは行わない
- **パネル D-15** は色票を**色相順に並べる**検査。**混同色線方向に交差線**が現れる
- **アノマロスコープ**は**赤緑の混色を黄に合わせる**精密検査で、**確定診断**に使う
- 検査の目的は「異常の診断」ではなく**「自分の色覚を知ること」**
- 学校健診での色覚検査は**2003 年に必須から削除**、その後**希望者への実施**が推奨されている
- 検査表は**75cm の距離・自然光に近い照明・3 秒以内**で読むのが基本
- 検査の仕組み（混同される色のパターン）は、混同しにくい配色設計の参考になる
