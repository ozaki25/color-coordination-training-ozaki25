# lesson23: 加齢にともなう眼疾病 — 高齢者の見え方を変える病気

## このレッスンで学ぶこと

- 高齢期に多い4大眼疾病（白内障・加齢黄斑変性・緑内障・糖尿病網膜症）を知る
- 各疾病が見え方をどう変えるか（色・視野・コントラスト）を理解する
- 早期発見・早期治療の重要性を知る
- 加齢変化（生理的）と眼疾病（病的）を区別する

## 加齢と眼疾病

[lesson21](/lessons/lesson21/)・[lesson22](/lessons/lesson22/) では「**誰にでも起こる加齢による視機能の変化**」を学びました。本レッスンでは、それとは別のカテゴリーとして「**一部の高齢者に起こる眼疾病**」を学びます。

両者は次のように整理できます。

| 区分 | 内容 | 例 |
|------|------|------|
| **加齢＝生理的変化** | 誰にでも起こる、年齢に伴う自然な変化 | 水晶体の黄変、瞳孔の縮小、視力低下、コントラスト感度の低下 |
| **疾病＝病理的変化** | 一部の高齢者に起こる、治療の対象となる病気 | 白内障、加齢黄斑変性、緑内障、糖尿病網膜症 |

実際には、加齢変化と疾病は**連続的に進行する**こともあり（例：水晶体の黄変が進むと白内障に移行する）、明確に分けにくい場合もあります。それでも「**加齢＝生理的変化／疾病＝病理的変化**」と区別して捉えておくと、配慮設計の整理がしやすくなります。

[lesson21](/lessons/lesson21/) で学んだ加齢一般の変化（水晶体の黄変・瞳孔の縮小・視力低下など）は、誰にでも起こる**生理的な変化**です。一方、ここで扱うのは**病気としての変化**で、放置すると失明にもつながる重要なものです。

代表的な4つの眼疾病は次の通りです。

| 疾病 | 主な部位 | 主な見え方の変化 |
|------|---------|---------------|
| 白内障 | 水晶体 | 全体が霞む、色がくすむ |
| 加齢黄斑変性 | 網膜の黄斑 | 中心が歪む・暗く欠ける |
| 緑内障 | 視神経 | 視野が周辺から欠ける |
| 糖尿病網膜症 | 網膜の血管 | 視力低下・出血による視野欠損 |

色のUDを進めるうえで、これらの疾病による見え方の違いを理解しておくことは、**高齢者・ロービジョン**（[lesson24](/lessons/lesson24/)）への配慮設計の基礎になります。

## 白内障

**白内障**は、眼の中の**水晶体が濁る**病気です。加齢でほとんどの人に起こり、80歳までに大半の人が何らかの程度で発症します。

### 見え方の変化

- 全体に**霞んで**見える
- 強い光がまぶしく感じる（**グレア**）
- 色が**くすみ**、特に青系が判別しにくい（水晶体が黄褐色に濁るため）
- 細部のコントラストが落ちる

### 治療

水晶体を取り除いて**人工の眼内レンズ**に置き換える手術が標準的な治療です。日本では年間100万件以上行われ、安全性が確立しています。

::: tip 白内障手術と色の見え方
白内障手術で水晶体を透明な眼内レンズに置き換えると、**抜けていた青系の波長が再び届く**ようになります。手術後に「世界が青く見える」と感じる人がいるのはそのためです。
:::

## 加齢黄斑変性

**加齢黄斑変性（AMD: Age-related Macular Degeneration）**は、**網膜の中心にある黄斑**が変性する病気です。黄斑には錐体が集中しており（[lesson07](/lessons/lesson07/) 参照）、**視力の中心**を担います。

### 見え方の変化

- 視野の**中心が歪んで**見える（**変視症**）
- 中心が**暗く欠ける**（**中心暗点**）
- 文字や顔の細部が見えない
- 周辺視野は保たれているため、移動には大きな支障が出ないことが多い

### タイプと進行

- **萎縮型**: 黄斑がゆっくり萎縮していくタイプ。進行が遅い
- **滲出型**: 異常な血管（新生血管）から液体や血液が漏れるタイプ。進行が速く視力低下が急

### 治療

滲出型には**抗VEGF薬の硝子体内注射**などの治療があります。早期発見が視力維持の鍵です。

::: warning 中心が見えないと「読む・顔を見る」が大きく制限される
黄斑変性では視野の中心がやられるため、文字が読めず、人の顔の見分けも難しくなります。色情報も中心視野で得ているため、**色の判別力も下がります**。
:::

## 緑内障

**緑内障**は、眼圧などの影響で**視神経が傷つき、視野が徐々に欠けていく**病気です。日本人の**失明原因の第1位**で、40歳以上の約5% に見られます。

### 見え方の変化

- **視野の周辺から**少しずつ欠けていく
- 進行が**ゆっくり**で、本人が気づきにくい
- 反対側の眼で補い合うため、片眼の異常を見逃しやすい
- 進行すると中心視野も侵され、最終的には失明することがある

### 治療

完全に元に戻す治療は**ない**ため、**眼圧を下げる点眼薬・手術**で進行を抑えるのが基本です。早期発見が極めて重要です。

::: info 緑内障と色の見え方
緑内障では、色そのものというより**コントラスト感度**が低下します。視野が欠ける範囲では情報が得られず、特に**青系の色**を見分ける力が早期から落ちることが知られています。
:::

## 糖尿病網膜症

**糖尿病網膜症**は、糖尿病の合併症として網膜の**血管が傷つく**病気です。日本人の**失明原因の上位**を占めます。

### 見え方の変化

- 血管から漏れた血液や水分で**視界が霞む・歪む**
- 出血や網膜剥離による**視野の欠損**
- 急激な視力低下が起こりうる
- 黄斑部にも影響が及ぶと**色やコントラスト**の感度が落ちる

### 進行と治療

進行段階に応じて**レーザー光凝固・硝子体手術・抗VEGF注射**などが選ばれます。何より**糖尿病自体のコントロール**（血糖管理）が予防・進行抑制の基本です。

## 4つの疾病が見え方に与える影響まとめ

| 疾病 | 視野の欠け方 | 色の見え方 | コントラスト |
|------|------------|----------|------------|
| 白内障 | 全体が霞む | 青系が弱まる | 全体に低下 |
| 加齢黄斑変性 | 中心が欠ける・歪む | 中心視野の色が判別できない | 中心で低下 |
| 緑内障 | 周辺から欠ける | 青系が早期から低下 | 全体に低下 |
| 糖尿病網膜症 | 部分的・不規則に欠ける | 黄斑が侵されると低下 | 全体に低下 |

これらの疾病は単独で起こることもあれば、**複数が併発する**こともあります。高齢者の中には複数の疾病を持つ方も多く、設計時には**最悪の場合でも情報が伝わる**配慮が必要です。

## 早期発見の重要性

これらの疾病は、共通して次の特徴を持ちます。

- **進行は遅く、初期は自覚症状がほとんどない**
- **失われた視機能は元に戻らない**ことが多い
- **早期に発見・治療を始めれば進行を抑えられる**

::: tip 定期的な眼科健診が鍵
40歳を過ぎたら、自覚症状がなくても定期的な眼科健診を受けることが推奨されます。「文字が見にくい」「色が違って見える」と気づいたときには、すでに進行している可能性があります。
:::

## 色のUDへの示唆

これらの疾病による見え方の変化は、**色覚特性とは別の方向**で色の見分けにくさを生みます。高齢者向け、あるいは公共向けの設計では次が重要です。

- **明度差・コントラストの確保**（白内障・緑内障対策）
- **青系の色だけに頼らない**（白内障・緑内障対策）
- **中心視野だけに情報を集中させない**（黄斑変性対策）
- **小さな文字・細い線を避ける**（あらゆる疾病・加齢対策）
- **色名や形状の併用**（色覚特性者と共通の対策）

詳細な配慮ポイントは [lesson24](/lessons/lesson24/) で扱います。

## 試験のポイント

- 4大疾病 = **白内障・加齢黄斑変性・緑内障・糖尿病網膜症**
- **白内障**は水晶体の濁り。**眼内レンズ手術**で治療
- **加齢黄斑変性**は**中心視野**が歪む・欠ける
- **緑内障**は**周辺視野**から欠け、自覚しにくい。**失明原因1位**
- **糖尿病網膜症**は血管障害で、**糖尿病のコントロール**が予防・進行抑制の基本
- 白内障・緑内障では**青系の色**が見えにくくなりやすい
- 一度失われた視機能は戻りにくく、**早期発見が決定的に重要**
- 高齢者は複数の疾病が**併発**することがあり、設計は**最悪条件**を想定する
