# lesson24: 高齢者・ロービジョンへの配慮 — 多くの人に届くデザイン

## このレッスンで学ぶこと

- **ロービジョン**の意味と全盲との違いを理解する
- 高齢者・ロービジョンに共通する配慮ポイントを理解する（明度差・文字サイズ・グレア・色以外の手がかり）
- **「高齢者に優しいデザインは多くの人に優しい」**というUDの原則を理解する
- 色のUDが幅広い人に届くものであることを再確認する

## ロービジョンとは

**ロービジョン（Low Vision）**とは、**視覚に障害があるが全盲ではない状態**を指します。眼鏡やコンタクトレンズで矯正しても日常生活・職業生活に支障がある程度の見え方です。

日本のロービジョン者は数十万人〜百万人規模と推計され、その多くは**高齢者**です。原因は [lesson23](/lessons/lesson23/) で学んだ眼疾病が中心で、白内障・加齢黄斑変性・緑内障・糖尿病網膜症が四大原因とされます。

ロービジョンの「見え方」は人によって大きく異なります。

| 見え方の例 | 主な原因 |
|----------|---------|
| 全体が霞む | 白内障・角膜混濁 |
| 中心が見えない | 加齢黄斑変性 |
| 視野が狭い（求心性視野狭窄） | 緑内障・網膜色素変性症 |
| 視野の一部が欠ける | 糖尿病網膜症・脳梗塞後 |
| まぶしさに弱い | 白内障・A型・網膜疾患 |

::: info 全盲とロービジョンの違い
**全盲**は視覚情報をほとんど利用できない状態、**ロービジョン**は視覚情報を**活用できる範囲がある**状態を指します。ロービジョン者には、見える範囲を最大限活かせるよう設計することが重要です。
:::

## 高齢者・ロービジョンに共通する配慮

加齢による変化（[lesson21](/lessons/lesson21/)・[lesson22](/lessons/lesson22/)）と眼疾病（[lesson23](/lessons/lesson23/)）は、ロービジョンと多くの配慮ポイントを共有します。

### 明度差・コントラストの確保

色の違いだけでなく、**明るさの差**で図と地を区別できるようにします。

- 文字と背景の**コントラスト比**はWCAG AA基準で **標準テキスト4.5:1以上**、**大きな文字・図形3:1以上**
- 白黒の最大コントラスト比は21:1
- 段差や手すりは床面・壁面と明度差をつける（つまずき防止）
- 操作ボタンと筐体の色を分け、輪郭を明示する

::: tip 明度差はあらゆる視覚障害に有効
色覚特性者は色相が見分けにくくても**明度差は感じられます**。白内障で霞んでいても、ロービジョンで視野が狭くても、明度差が大きければ図と地が分離します。**明度差**は色のUDで最も汎用的な対策です。
:::

### 文字サイズと書体

- **文字は十分な大きさ**にする（公共サインは想定距離に合わせて算出）
- **読みやすい書体**を選ぶ（ゴシック体・UDフォントが基本。極端な装飾書体は避ける）
- **太さは中字以上**にして線が消えないようにする
- 文字間・行間に**ゆとり**を持たせる
- **長文は短文・箇条書きに分解**する

<!-- textlint-disable ja-technical-writing/ja-unnatural-alphabet -->

UDフォントは、似た文字（数字の0と英大文字のO、数字の6と英小文字のbなど）の判別性を高めた書体です。高齢者・ロービジョン・ディスレクシアなど幅広い人に読みやすいことが評価されています。

<!-- textlint-enable ja-technical-writing/ja-unnatural-alphabet -->

### グレア（まぶしさ）対策

加齢で水晶体が混濁すると、入った光が眼内で散乱し**まぶしさ**が増します。白内障やA型・網膜疾患でも同様です。

- 直接光源が見える位置に**強い裸電球**を置かない
- ガラス・床・壁・印刷物の**反射**を抑える（つや消し仕上げ）
- 屋外サインは**逆光**を避ける位置に
- 透明アクリルに印刷物を入れると反射で読めなくなることがある
- 屋外から屋内に入る場所は**明暗の急変**を緩和する（前室で明るさを段階的に下げる）

### 色覚と無関係に意味が伝わる手がかり

色情報が伝わらなくても**情報そのものは届く**ようにします。

- **形**: 信号機の形違い、エラー時に「✕」マーク
- **記号・アイコン**: 注意喚起マーク、ピクトグラム
- **位置**: 信号機の左右、レイアウトの並び順
- **テキスト・色名**: 「赤：危険」「青：安全」と書き添える（[lesson12](/lessons/lesson12/)・[lesson17](/lessons/lesson17/) 参照）
- **音声**: 駅の発車音、家電のビープ、音声ガイド
- **触覚**: 点字、シャンプー容器のギザギザ、紙幣の識別マーク

::: warning 色だけに頼らないことがUDの出発点
「赤いボタンを押してください」「黄色いラインに沿って進んでください」のように**色だけが手がかり**になっていないかを必ず確認します。ロービジョン・色覚特性者・暗い場所にいる人・モノクロ印刷の利用者など、誰かは確実に困ります。
:::

### 操作のしやすさ

視覚以外でもアクセスできるようにすると、ロービジョンや高齢者にとって決定的に便利になります。

- 大きな**物理ボタン**（タッチパネルだけにしない）
- 操作部に**触って分かる目印**
- **音声フィードバック**（押したら鳴る）
- **エラー時の音と表示**を併用

::: tip 服薬支援の「一包化」
高齢者が複数の薬を服用する場面では、薬局で1回分の薬をひとつの袋にまとめる**一包化（いっぽうか）**という仕組みが使われます。色や形で似た錠剤を見分ける負担を減らし、飲み忘れ・飲み間違いを防ぐ、視覚・認知の両面に効く生活密着型のUD事例です。
:::

## 「高齢者に優しいデザインは多くの人に優しい」

UDの出発点は、特定の人を「特別扱い」するのではなく、**最初から多くの人が使えるように設計する**ことです（[lesson01](/lessons/lesson01/) 参照）。

加齢による変化は誰にでも訪れます。高齢者向けに行った配慮は、結果として次の人々にも届きます。

| 配慮 | 直接の対象 | 同時に助かる人 |
|------|----------|--------------|
| 明度差・コントラスト | 高齢者・ロービジョン | 色覚特性者・暗所利用者・モノクロ印刷 |
| 大きな文字 | 高齢者・ロービジョン | 子ども・遠くから見る人・タブレット利用者 |
| 色名の付記 | 色覚特性者 | 高齢者・ロービジョン・モノクロ印刷 |
| 形・記号の併用 | 色覚特性者 | 高齢者・外国人・幼児 |
| 音声フィードバック | ロービジョン・全盲 | 高齢者・忙しい人・暗所利用者 |
| 段差の明度差 | 高齢者・ロービジョン | 荷物を持つ人・夜間の利用者 |
| グレア対策 | 高齢者・ロービジョン | 屋外で見る人・コントラスト感度の低い人 |

これが**「高齢者に優しいデザインは多くの人に優しい」**と言われる理由です。色のUDは色覚特性者だけのものではなく、社会のあらゆる人に届くデザインの一部です。

## 設計の出発点としての姿勢

配慮を後から「付け足す」のではなく、設計の最初から組み込むことがUDの基本です。

- 色を選ぶときに**明度差**を最優先に確認する
- 文字を入れるときに**サイズ・書体**を最初に決める
- 情報を伝えるときに**色以外の手がかり**を必ず添える
- 設計後に**色覚シミュレーション**やモノクロ表示で確認する（[lesson27](/lessons/lesson27/)）

色のUDの設計手順は [lesson26](/lessons/lesson26/)、チェック方法は [lesson27](/lessons/lesson27/)、設計と修正の4ポイントは [lesson28](/lessons/lesson28/) で扱います。

| 用語 | 説明 |
|------|------|
| ロービジョン | 視覚障害があるが全盲ではない状態。見え方は人により多様 |
| コントラスト比 | 図と地の明るさの比。WCAG AAは標準テキスト4.5:1、大きな文字3:1以上 |
| UDフォント | 文字判別性を高めた書体。高齢者・ロービジョンなど幅広い人に読みやすい |
| グレア | まぶしさ。光源や反射光が視覚情報を奪う状態 |
| 色以外の手がかり | 形・記号・位置・テキスト・音声・触覚など。色覚と無関係に意味を伝える要素 |
| マルチモーダル | 視覚・聴覚・触覚など複数の感覚で情報を伝える設計 |

## 試験のポイント

- **ロービジョン = 視覚障害だが全盲ではない状態**。見え方は人によって大きく異なる
- 共通する配慮 = **明度差・文字サイズ・書体・グレア対策・色以外の手がかり**
- 文字色と背景のコントラスト比は **標準4.5:1、大きな文字3:1**（WCAG AA）
- **明度差**は色覚特性者・高齢者・ロービジョンすべてに有効な汎用対策
- 色だけに頼らず、**形・記号・位置・テキスト・音声・触覚**を組み合わせる
- グレア対策（反射抑制・直接光源の遮蔽・逆光回避）は加齢・疾病の両方に有効
- **「高齢者に優しいデザインは多くの人に優しい」**がUDの基本姿勢
- 配慮は後付けではなく、設計の**最初から**組み込む
